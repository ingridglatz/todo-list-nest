# To do list em NestJs

## Descrição

Projeto de exemplo para demonstrar a utilização do NestJs.

## Objetivo

Criar uma API para gerenciar uma lista de tarefas.

## Instalação

```bash
$ npm install
```

## Execução

```bash
# development
$ npm run start:dev
```

## Tecnologias

- NestJs
- Docker
- Prisma
- PostgreSQL
- Jest
